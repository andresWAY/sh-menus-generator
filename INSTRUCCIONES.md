# Guía de Uso Fácil - Generador de Menús

He detectado problemas con tu configuración de Python, así que he creado unos "accesos directos" para que todo funcione sin comandos complicados.

## 1. Instalación Automática

1. Ve a la carpeta del proyecto.
2. Haz doble clic en el archivo **`install.bat`**.
   - Se abrirá una ventana negra que instalará todo lo necesario.
   - Si dice "Todo listo", cierra la ventana.

## 2. Colocar tus Archivos

Ahora verás que han aparecido las carpetas. Llénalas con tus archivos:

- **`assets/fonts/`**: Copia aquí tus 3 fuentes `.ttf`.
- **`assets/backgrounds/`**: Copia aquí tus imágenes `.jpg` (ej: `ENSALADAS.jpg`).
- **`assets/static/`**: Copia aquí tu `ALERGENOS.pdf`.
- **`data/`**: Aquí está el `menu_data.xlsx` de ejemplo. Puedes editarlo o reemplazarlo.

## 3. Ejecutar el Programa

Cada vez que quieras usar la herramienta:

1. Haz doble clic en **`run_app.bat`**.
2. Se abrirá automáticamente la aplicación en tu navegador.
